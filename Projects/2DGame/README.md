# Game Development

Develop a 2D Game.
- Create specifications on what the game must support.
- Choose a development framework
- Select key systems
- Create extensions

## Gameplay Specifications

| Goal                                                | Feature                                                               |
| --------------------------------------------------- | --------------------------------------------------------------------- |
| Create a persistent 2D world with survival elements | Biomes, Day/Night, Weather, Resources, buildable homes.               |
| Exploration & Combat Progression                    | Party system, Quests, Items, Skills, Classes, Factions, Ranking       |
| Loot Progression                                    | Item Upgrades, Iconic Weapons, Ranked chests                          |
| Flexible character specialization                   | Inheritance-based class system, Skill-Styles (Physical, Mental, ... ) |
| (Un)Cooperative Gameplay                            | Scalable Networking (PvPvE)                                           |
| Bestiary and lore system                            |                                                                       |
