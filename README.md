## SDLC-Models

Serves as guideline to learn about topics on software development life cycles.



### Documentation/Postgres

1. **Specifications**: Understand the what and how of building software\
Depicts the process from idea to application, covering Requirements Engineering and foundational Development Frameworks (like Agile/Scrum).
2. **Software Testing**: Learn about quality assurance and finding defects.\
Details the Quality Assurance methodologies, from Test-Driven Development (TDD) and various Testing Strategies (e.g., Black-Box, White-Box) to advanced Security Tests and Defect Management.
3. **XaaS**: Understand modern deployment targets and environment automation\
Shows modern deployment environments, including the core concepts of cloud computing models (IaaS, PaaS, SaaS), Virtual Machines, Containers, and large-scale Orchestration (Kubernetes). This directory also covers the automation of these environments using Infrastructure as Code (IaC).
4. **DeveloperOperations**: See how all previous steps are automated into a reliable, continuous pipeline.\
Focuses on the automation practices that tie development and operations together, specifically detailing the stages of the CI/CD Pipeline (Continuous Integration, Continuous Delivery, Continuous Testing) and essential real-time application and infrastructure monitoring (Continuous Monitoring).
